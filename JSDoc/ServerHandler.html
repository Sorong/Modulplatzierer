Class: ServerHandler

ServerHandler

new ServerHandler(url)

Konstruiert ein ServerHandler-Objekt, dass die Kommunikation
mit der REST-Schnittstelle des als Parameter angegebenen Servers übernimmt.

Parameters:
Name Type Description
url string

Die URL des Servers mit dem kommuniziert werden soll.
Properties:
Name Type Description
serverUrl string

Die URL des Servers.
errorFunction function

Eine Standardfunktion, die im Fehlerfall ausgeführt wird.
